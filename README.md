# Langchain-LangGraph-LangSmith

A comprehensive repository for learning and implementing LangChain, LangGraph, and LangSmith, ranging from foundations to production-ready patterns.

## Repository Structure

### 📁 [foundations](./foundations/)

Core learning materials and exercises for the three pillars of the ecosystem.

- **[langchain/](./foundations/langchain/)**: Academy notes and foundational exercises.
  - `concepts.md`: Deep dive into LCEL, Runnables, and core abstractions.
  - `notes.md`: Practical implementation notes and tips.
  - **[lca-lc-foundations-main/](./foundations/langchain/lca-lc-foundations-main/)**: Official LangChain Academy notebooks covering foundational models, prompting, tools, memory, and more.
- **[langgraph/](./foundations/langgraph/)**: Advanced orchestration and state management.
  - `concepts.md`: Understanding State, Nodes, Edges, and Persistence.
  - `notes.md`: **Comprehensive Lesson-by-Lesson Notes** (Module 1: Fundamentals, Module 2: State and Memory) covering graph construction, tools, chains, routers, and LangSmith Studio integration.
  - `main_langraph_instructions.md`: **Crucial** best practices and rules for building graphs.
  - **[langchain-academy-langgraph/](./foundations/langgraph/langchain-academy-langgraph/)**: Comprehensive course modules (0-6).
    - **Module 1**: Basics, Routers, and Agents.
    - **Module 2**: Advanced State Management (including `state-schema.ipynb`), Persistence, and Memory.
    - **Modules 3-6**: Human-in-the-loop, Parallelization, and Deployment.
- **[langsmith/](./foundations/langsmith/)**: Observability, testing, and evaluation.

### 📁 [projects](./projects/)

Custom projects built using the foundations learned in the repository.

### 📁 [production-patterns](./production-patterns/)

Implementations of advanced patterns and articles for production-grade AI agents.

### 📁 [python_concepts_references](./python_concepts_references/)

- `LangGraph_Python_OOP.md`: A reference guide on how Python OOP concepts (Runnables, Pydantic, TypedDict) are applied within LangGraph.
- `classmethod_in_python.md`: Detailed explanation of `@classmethod` and its practical uses in Python development.
